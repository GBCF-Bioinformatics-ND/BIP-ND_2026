Bioinformatics Introductory Program (BIP)

2026 Bioinformatics Introductory Program (BIP) for Graduate Students

Foundations for wet-lab researchers: shell and HPC literacy, reproducible environments, public-database fluency, and a practical IDE workflow.

Format In person or hybrid
Duration Two days
Audience Graduate students, postdocs, research staff, faculty
Prerequisite No prior programming required

General Information

Back to top

Who

Mixed-skill learners working in life sciences who want a practical and reproducible bioinformatics workflow.

Where

Local HPC plus local VS Code via Remote-SSH. Delivery can be classroom, online, or hybrid.

What Learners Will Be Able To Do

  • Navigate an HPC and manipulate genomic files from the shell.
  • Create and use reproducible project environments.
  • Retrieve genomic data from web portals and APIs.
  • Use VS Code debugging, linting, formatting, and Git integration.

Accessibility

Request accommodations before the workshop. Instructors can adapt pacing, font sizes, break structure, and delivery format.

See also the instructor checklist and notes in the docs folder.
